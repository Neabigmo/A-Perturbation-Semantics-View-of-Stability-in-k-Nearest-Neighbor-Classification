# Archive

This folder holds retired or historical material so the active repository stays
easy to scan.

## What Lives Here

- `archive/legacy/`: files that used to sit in tracked docs or task folders
  before the project was refocused around `main.pdf`
- `archive/drafts/`: alternate paper drafts and their build leftovers
- `archive/build/`: older LaTeX outputs and logs
- `archive/notes/`: review notes, planning notes, and similar side documents
- `archive/assets/`: example figures and other moved media
- `archive/tools/`: local helper files that are no longer part of the active
  working set

The active working set is the repository root plus `paper/`, `src/`,
`experiments/`, `outputs/`, and `tests/`.
